# Hubot Wttr

**Hubot script that asks for current weather to Wttr.in and show the result**

### Install:

````
$ npm install hubot-wttr --save
````

Add to `external-scripts.json`:

````
[
  ..
  "hubot-wttr"
]
````

### Use:

````
hubot> hubot weather
````

**Example:**

````
hubot> hubot weather Santiago, Chile

hubot>
Weather for City: Santiago, Chile

               Overcast 
      .--.     17 °C          
   .-(    ).   ↖ 7 km/h       
  (___.__)__)  8 km           
               0.1 mm         

````
